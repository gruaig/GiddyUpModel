Key Locations
============

Daily Workflow:
  Morning:  ./scripts/morning_prep.sh
  Start:    sudo systemctl start giddyup-bot
  Evening:  ./scripts/end_of_day.sh

Documentation:
  Main:     README.md
  Docs:     docs/README.md
  Quick:    docs/QUICK_START_SERVICE.md

Configuration:
  Betfair:  HorseBot_config.py (create from template)
  Telegram: telegram_config.py (create from template)

Logs & Output:
  Bets:     strategies/logs/daily_bets/
  Reports:  strategies/logs/automated_bets/
  Tweets:   strategies/logs/tweets/

Test:
  Telegram: python3 telegram_bot.py
  Bot:      python3 HorseBot_Simple.py start $(date +%Y-%m-%d) 5000
